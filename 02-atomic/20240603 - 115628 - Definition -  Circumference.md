A circumference is defined as the lugar geométrico to which belong: 
+ **All points with the same distance r to a fixed point c**
	+ r → radius
	+ c = ($x_0,y_0$) → center

It is described by: 
$$
\boxed{(x-x_0)^2 + (y - y_0)^2 = r^2: r>0}
$$
