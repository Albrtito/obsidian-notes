# decision trees for regression
> [!NOTE] Intro: 
> How can we use the basic idea of decision trees for regression?

The main change is that the leafs **wont carry a class value but one sinlge value as the mean of all values in that leaf**. 
Impurity then is meassuerd as variance from the values of that leaf to the value assigned. Variance|MSE to the mean value of the interval.
