# reduction of the number of comparisons in ARM
> [!info] Intro: 
> Counting the support of a candidate means comparing it to other ones. Because there are so many candidates, if somehow we could **not count the support of some candidates** the mining would be optimized.

To do so we'll first have to create a **hash map** with all the possible candidates.
