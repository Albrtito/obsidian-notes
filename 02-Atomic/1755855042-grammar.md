# grammar
> [!NOTE] Intro: 
> A grammar expresses a languages structure. Defined as a quadruple
> $$
> G = (N, \sum, S, P)
> $$
> - **N:** set of non-terminal symbols
> - **$\sum$**: Set of terminal symbols
> - **S $\in$ N** Start symbol of the grammar
