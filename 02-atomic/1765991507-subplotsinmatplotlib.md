# subplots in matplotlib
> [!info] Intro: 
> Within matplotlib is really usefull to create one image with several plots, obtaining a general view with lots of info. 
> In order to do so the `plt.subplots()` method is used. 
> A tuple with a figure object and an array of axis objects is returned. One axis object per plot.

This method creates a distribution of plots based on: 
- Number of horizontal plots
- Number of vertical plots 
- If they share or not x or y axis 

Each of this plots is then can be modified by using its axis object.
