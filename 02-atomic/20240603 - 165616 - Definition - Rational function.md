Rational functions, or quotient functions have the form: 
$$
f(x) = \frac{a_0 + a_1x^1 + a_2 x^2 + ... + a_nx^n }{a_0 + a_1x^1 + a_2 x^2 + ... + a_nx^n }
$$
**Properties:**
+ $Dom = \{\text{ denominator different from 0 }\}$
