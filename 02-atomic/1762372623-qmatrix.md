# q-matrix
> [!info] Intro: 
> The Q-Matrix is used to save all the values for the Q function of a [[1762205273-temporaldifferencelearning|temporal difference learning]] model. The Q-Values are learned based on some policy with high exploration and then when the training is done the agent will just choose the path trough the graph with the highest reward.
