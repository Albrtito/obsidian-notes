# Bayesian Information Criterion
> [!info] Intro: 
> Used as well as the [[1762710178-akaikeinformationcriterion|AIC]] to evaluate a regression model based on the fit and the number of variables it uses. **A lower value is better**

$$
BIC = k \cdot \ln(n) - 2\ln \text{(likelihood)}
$$
**where:**
 - k -> Number of parameters
 - n -> Number of samples
