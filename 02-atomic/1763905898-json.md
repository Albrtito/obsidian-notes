# json
> [!info] Intro: 
> Lightweight data-interchange language. Used to save data in an structured way. Easy to read (human readable). 

It defines objects as a pair of `string:value` enclosed by brackets. Then values can be objects by themselves creating complex structures. 
All data types and documentation on json can be foundon the oficial web[^1]
