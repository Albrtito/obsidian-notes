# Akaike Information Criterion
> [!info] Intro: 
> How good is the model for the number of variables used and the final fit obtained. The lower the better. This coefficient penalizes models that use more variables. 

$$
AIC = 2K - 1\ln \text{(likelihood)}
$$
**where:**
 - k -> The number of parameters used 
 - likelihood -> Related to how good the fit is
